# Microsoftアカウントの作成・ログイン方法

Minecraft 統合版でオンラインプレイをするには、**Microsoftアカウント（Xboxアカウント）**が必要です。

以下では、年齢・使用端末ごとにアカウント作成とログイン手順を案内します。

---

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

## 1. アカウントの作成方法

<Tabs queryString="accountType" className="tabs-account-type">
  <TabItem value="adult" label="成人の場合" default>

### 👤 成人（18歳以上）の場合

👉 [Microsoft アカウント作成ページ](https://signup.live.com/)

#### ✅ 作成手順

1. 上記のリンクを開く  
2. メールアドレスを入力（または「新しいメールアドレスを取得」）  
3. パスワード、名前、生年月日を入力（生年月日は2006年以前を推奨）  
4. セキュリティ確認（画像認証）

これでアカウント作成完了です。

:::info
成人アカウントは制限が少なく、すぐにオンラインプレイが可能です。
:::

  </TabItem>
  <TabItem value="minor" label="未成年の場合">

### 🧒 未成年（13歳～17歳）の場合

👉 [Microsoft アカウント作成ページ](https://signup.live.com/)

#### ✅ 作成手順

1. 上記のリンクを開く  
2. メールアドレスを入力  
3. 生年月日を「2007年以降」で入力（未成年と判定）  
4. 保護者の Microsoft アカウントのメールアドレスを入力するよう求められる  
5. 保護者に「Microsoftファミリーアカウント」に追加される  
6. 保護者が [ファミリー設定ページ](https://account.microsoft.com/family/) でオンラインプレイを許可

:::caution
保護者の承認がないと、サーバーへの接続やフレンド機能が制限されます。
:::

  </TabItem>
</Tabs>

## 2. Minecraftへのログイン方法（端末別）

<Tabs queryString="device" className="tabs-login-device">
  <TabItem value="pc_iphone" label="Windows / iPhone" default>

### 💻 Windows10/11 または 📱 iPhone・iPad

1. Minecraft を起動  
2. 右下の「サインイン（Microsoft アカウント）」をタップ  
3. 表示されたコードを確認（例: XXXX-XXXX）  
4. 別の端末で [https://aka.ms/remoteconnect](https://aka.ms/remoteconnect) を開く  
5. 表示されたコードを入力し、Microsoftアカウントにログイン  
6. Minecraftに自動で戻り、「ようこそ ○○さん」と表示されれば完了！

:::info
一度ログインすれば、以降のログイン操作は不要です。
:::

  </TabItem>
  <TabItem value="console" label="Switch / PS4/5 / Xbox">

### 🎮 Switch / PS4 / PS5 / Xbox

1. Minecraft を起動  
2. 「サインイン」または「Microsoftアカウントに接続」を選択  
3. 表示されたコードを確認（例: XXXX-XXXX）  
4. スマホやPCで [https://aka.ms/remoteconnect](https://aka.ms/remoteconnect) にアクセス  
5. コードを入力 → Microsoftアカウントでログイン  
6. 成功すると、ゲームに戻ってオンライン機能が有効になります

:::caution
SwitchやPSでは、**Nintendo OnlineやPlayStation Plusの加入が必要な場合があります。**
:::

  </TabItem>
</Tabs>

## 3. ログイン後にできること

- ✅ フレンド機能の使用
- ✅ 外部サーバー（24sanなど）への接続
- ✅ ゲーマータグ（プレイヤー名）の設定
- ✅ チャットやマルチプレイ機能の使用

---

## 4. よくあるトラブルと対処法

| 症状 | 対処法 |
|------|--------|
| リモート接続コードが出ない | 一度ゲームを再起動 |
| `aka.ms/remoteconnect` にアクセスできない | 別のブラウザ・端末で試す |
| オンラインに接続できない | 保護者アカウントの制限を確認 |
| プレイヤー名が変わらない | [Microsoftアカウントの管理](https://account.microsoft.com/) から変更可能 |
