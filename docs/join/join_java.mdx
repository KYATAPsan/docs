# Java版での参加方法

Minecraft Java Edition（PC版）をお使いの方向けに、**24sanサーバーへの参加手順**をご案内します。

:::info

24sanサーバーは Java版・統合版の両方に対応しています。  
このページは Java版プレイヤー向けの説明です。

:::

---

## 1. 必要な環境

✅ 参加するには以下の環境が必要です：

- Minecraft Java Edition の正規アカウント  
- 対応バージョン（例：1.20.x〜1.21.5）  
- インターネット接続

:::caution

一部のMODは制限されています。  
[利用可能MOD一覧](/docs/rules/allowed_mods) を必ずご確認ください。

:::

---

## 2. サーバー情報

| 項目 | 内容 |
|------|------|
| サーバーアドレス | `24san.org` |
| ポート | 不要（自動設定） |
| 対応バージョン | 1.13 ～ 1.21.5 |

---

## 3. 参加手順

🎮 **Minecraft ランチャーでの操作手順**

1. **Minecraft ランチャーを起動**  
2. 「インストール」から `1.21.5` バージョンを選択して起動  
3. 「マルチプレイ」を選択  
4. 「サーバーを追加」をクリックし、以下を入力：  
   - サーバー名：`24san`（自由に設定可）  
   - サーバーアドレス：`24san.org`  
5. 「完了」でサーバーを一覧に登録  
6. サーバーを選んで「接続」！

---

## 4. よくあるトラブルと対処法

| 症状 | 対処方法 |
|------|----------|
| `旧バージョンで接続できない` | バージョンを `1.21.5` に変更してください |
| `MODを入れて重い` | [Sodium](https://www.curseforge.com/minecraft/mc-mods/sodium) など軽量化MODを検討してください |
| `エラーで接続できない` | アドレス `24san.org` を正しく入力したか確認してください |

:::caution

ファイアウォールやセキュリティソフトがブロックしている場合もあります。接続できない場合は一時的に確認してください。

:::
