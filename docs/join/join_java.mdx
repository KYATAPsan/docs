# Java版での参加方法

Minecraft Java Edition（PC版）をお使いの方向けに、**24sanサーバーへの参加手順**をご案内します。

---

## 1. 必要な環境

- Minecraft Java Edition 正規アカウント
- 対応バージョン（例：1.20.x）
- インターネット接続

:::info

一部のMODは制限されています。  
[利用可能MOD一覧](/docs/rules/allowed_mods) をご確認ください。

:::

---

## 2. サーバー情報

| 項目 | 内容 |
|------|------|
| サーバーアドレス | `24san.org` |
| ポート | 不要（自動） |
| 対応バージョン | 1.13 ~ 1.21.5 |

---

## 3. 参加手順

### 🎮 Minecraft ランチャーでの操作

1. **Minecraft ランチャーを起動**  
2. 上部の「インストール」から `1.21.5` バージョンを選択して起動  
3. 「マルチプレイ」を選択  
4. 「サーバーを追加」をクリック  
   - サーバー名：`24san`（任意）  
   - サーバーアドレス：`24san.org`  
5. 「完了」を押して一覧に登録  
6. サーバーを選んで「接続」！

---

## 4. よくあるトラブルと対処法

| 症状 | 対処方法 |
|------|----------|
| `旧バージョン` で接続できない | 最新の 1.21.5 に切り替えてください |
| MODを入れて重い | [Sodium](https://www.curseforge.com/minecraft/mc-mods/sodium) などの軽量化MODを検討してください |
| エラーで接続できない | アドレスが `24san.org` であるか確認してください |

---

## 5. サーバーのルール確認

参加する前に、以下のページも必ずご確認ください：

- [利用規約](/docs/rules/terms_of_use)
- [プライバシーポリシー](/docs/rules/privacy_policy)
- [利用可能MOD一覧](/docs/rules/allowed_mods)
