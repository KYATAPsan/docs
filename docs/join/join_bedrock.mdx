# 統合版での参加方法（Minecraft Bedrock Edition）

Minecraft 統合版（Bedrock Edition）をお使いの方向けに、**24sanサーバーへの参加手順**をご案内します。  
iOS、Android、Switch、PS4、Windows10版などが対象です。

:::info

このページでは「アドレス接続」と「フレンド経由接続」の両方の方法を紹介します。  
どちらか使いやすい方法を選んでください。

:::

---

## 1. 必要な環境

✅ 参加には以下の条件が必要です：

- Minecraft Bedrock Edition（モバイル・Switch・Win10版など）
- インターネット接続
- 最新バージョン（例：1.20.x〜）

---

## 2. サーバー情報

| 項目 | 内容 |
|------|------|
| サーバーアドレス | `24san.org` |
| ポート番号 | `19132`（通常のデフォルト） |
| 対応バージョン | 1.20.x 以降 |

:::caution

ポート番号を間違えると接続できません。**19132** を必ず設定してください。

:::

---

## 3. 方法①：アドレスから接続する

📱 Nintendo Switch / スマホ / Windows10 向けの方法です。

### 🔧 手順

1. Minecraft を起動し、「プレイ」→「サーバー」→「サーバーを追加」へ進む  
2. 以下の情報を入力：

| 設定項目 | 入力内容 |
|----------|-----------|
| サーバー名 | `24san`（任意） |
| サーバーアドレス | `24san.org` |
| ポート番号 | `19132` |

3. 「保存」または「完了」を押す  
4. 登録したサーバーを選んで「参加」！

---

## 4. 方法②：フレンド経由で参加する

🎮 Xbox Live にログインした状態で、運営スタッフとフレンドになることで参加可能です。

### 🤝 フレンド申請手順

1. Microsoft アカウントにログイン（Xbox Live）  
2. 運営のゲーマータグ（例：`24san_admin`）にフレンド申請を送信  
3. フレンド承認後、「フレンド」タブから `24san_admin` のワールドに参加

:::info

フレンド参加は手動承認のため、タイミングによっては反映まで時間がかかる場合があります。

:::

---

## 5. よくあるトラブルと対処法

| 症状 | 対処方法 |
|------|----------|
| 「サーバーに接続できません」 | アドレスやポートの入力ミスがないか再確認してください |
| Switchでサーバー欄が出ない | 【BedrockTogether】【MC Server Connector】などの中継アプリを使用してください |
| 英語表示になる | ゲーム設定の言語を「日本語」に変更してください |
