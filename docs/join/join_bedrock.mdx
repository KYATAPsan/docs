# 統合版での参加方法（Minecraft Bedrock Edition）

Minecraft 統合版（Bedrock Edition）ユーザー向けに、**24sanサーバーへの参加方法**を以下の2種類から選べます：

- アドレスからの直接接続
- フレンド経由での接続（Xbox Live）

:::info

Switch、スマホ、Windows10版など、統合版プレイヤーはこのページを参考にしてください。

:::

## ✅ 方法①：アドレスから接続する

📱 Nintendo Switch / スマホ / Windows10 向けの方法です。

### 🔧 手順

1. Minecraft を起動し、「プレイ」→「サーバー」→「サーバーを追加」
2. 以下の情報を入力：

| 設定項目 | 入力内容 |
|----------|-----------|
| サーバー名 | `24san`（任意） |
| サーバーアドレス | `24san.org` |
| ポート番号 | `19132`（※忘れずに！） |

3. 「保存」または「完了」→ 登録したサーバーを選んで「参加」

:::caution

ポート番号を `19132` に設定していないと接続できません。ご注意ください。

:::

## ✅ 方法②：フレンド経由で接続する

🎮 Xbox Live にログインし、運営とフレンドになることで参加できます。

### 🤝 フレンド申請手順

1. Microsoft アカウントで Xbox Live にログイン  
2. 運営のゲーマータグ `24san_admin`（仮）にフレンド申請を送る  
3. 承認後、「フレンド」タブに `24san_admin` が表示されるので、そこから参加！

:::info

フレンド経由の参加は、承認まで時間がかかる場合があります。  
事前に Discord などで連絡をするとスムーズです。

:::

## よくあるトラブルと対処法

| 症状 | 対処方法 |
|------|----------|
| サーバーに接続できない | アドレス・ポートの入力ミスがないか確認してください |
| Switchでサーバー欄が表示されない | 中継アプリ（例：BedrockTogether）を使用してください |
| 英語表示になる | 設定から日本語に変更してください |
