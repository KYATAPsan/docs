# 24sanサーバー 利用規約

:::info

本利用規約（以下、「本規約」といいます）は、Minecraft マルチプレイサーバー「24sanサーバー」の提供および利用に関する条件を定めるものであり、本サーバーを利用するすべてのプレイヤーに適用されます。

本サーバーに接続しプレイを開始した時点で、本規約に同意したものとみなされます。

:::

---

## 1. 禁止事項

プレイヤーは、以下に該当する行為を行ってはなりません：

- Minecraft公式のEULAその他関連法令に違反する行為  
- 不正なクライアント、チートツールの使用  
- サーバーに対するDDoS攻撃、意図的なラグ発生  
- 荒らし、不適切なスキン・ユーザー名の使用  
- スパム、暴言、差別的発言、公序良俗に反する行為  
- 当サーバーやプレイヤーに対する誹謗中傷・嫌がらせ行為  
- 他サーバーの無断宣伝・勧誘（知人経由を含む）  
- バグ・仕様の悪用  
- スタッフのなりすまし  
- その他、運営が秩序を損なうと判断した行為

:::caution

明確に記載されていない行為であっても、運営の判断により不適切とみなされる場合は警告やBAN等の対象になります。

:::

## 2. 制限・処罰

違反者には以下の処罰を行う場合があります：

- 警告  
- ミュート（チャット制限）  
- 操作制限（設置・破壊制限）  
- アカウントBAN（無期限）

:::danger

処罰は予告なく行われることがあり、解除は運営の再判断に基づきます。

:::

## 3. プレイヤーデータの取り扱い

以下の情報を取得・保管する場合があります：

- ユーザー名、UUID、IPアドレス  
- チャットログ、ログイン履歴  
- 操作記録・違反履歴など  
- その他運営上必要な情報

:::info

取得した情報は、安全運営・ログ管理・違反対策の目的に限って使用され、第三者には提供されません。

:::

## 4. サーバー内の制限

以下の建築や行動は制限または削除の対象になることがあります：

- 高負荷を与える装置（クロック回路、MOBトラップ等）  
- 他人の建築への侵食・妨害  
- 環境に悪影響を与えると判断される行為

:::caution

運営は事前の通告なく撤去・巻き戻し（ロールバック）を行うことがあります。

:::

当サーバーは以下の損害について一切の責任を負いません：

- 利用不能による逸失利益・データ損失

- 停止・メンテナンス等による影響

:::info

責任が発生する場合でも、最大で過去12ヶ月に支払われた金額が上限となります。
