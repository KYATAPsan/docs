# 利用可能MOD一覧

当サーバーでは、プレイヤーの快適性を高める目的で、以下の一部MODの使用を許可しています。  
ただし、**サーバーに負荷を与えたり不正行為に繋がるMODは禁止**となります。

:::caution

使用可能なMODであっても、今後のアップデートやトラブル報告により禁止される可能性があります。  
使用は自己責任でお願いいたします。

:::
