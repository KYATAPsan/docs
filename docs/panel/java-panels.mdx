# Javaパネルの使い方

24sanサーバーでは、Java版プレイヤー向けに複数のGUIパネルが提供されています。  
ここでは、各パネルの使い方を切り替えて確認できます。

---

import Tabs from '@theme/Tabs';

## 🧭 メインパネル

すべてのパネルの起点となるパネルです。  
以下の機能にアクセスできます：

- ギルド管理
- テレポート機能
- ポイントショップ
- 通報システム
- PvP / PvE 設定切り替え

📌 **開き方：**  
- コンパスを右クリック  
- または `/cp main` コマンドを実行
  </div>
  </TabItem>
  <TabItem value="guild" label="ギルドパネル">

## 🛡️ ギルドパネル

ギルドの作成・参加・管理ができるパネルです。

- ギルドの作成（名前・カラー・公開設定）
- ギルドメンバーの確認
- ギルドへの参加 / 脱退

📌 **開き方：**  
メインパネル → ギルドパネル

  </TabItem>
  <TabItem value="teleport" label="テレポートパネル">

## 🧭 テレポートパネル

プレイヤーの移動をサポートするパネルです。

- スポーン地点へ移動
- ギルド本拠地へ移動
- ワープポイントへの移動（許可された地点のみ）

📌 **開き方：**  
メインパネル → テレポートパネル

  </TabItem>
  <TabItem value="shop" label="ショップパネル">

## 🛒 ショップパネル

ポイントを使用してアイテムやサービスを購入できます。

- アイテム購入（ツール・装備など）
- 特別なサービス（ワープ券・称号など）

📌 **開き方：**  
メインパネル → ショップパネル  
💰 所持ポイントは `%pointshop%` で確認できます。

  </TabItem>
  <TabItem value="report" label="通報パネル">

## 🚨 通報パネル

荒らしや不具合などを簡単に報告できるパネルです。

- 荒らし / バグ報告フォーム
- 報告した内容の履歴確認
- 通報に対する返信確認

📌 **開き方：**  
メインパネル → 通報パネル  
💬 または `/rpt create` コマンドでも通報可能です。

  </TabItem>
  <TabItem value="versu" label="PvP-PvEパネル">

## ⚔️ PvP / PvE 切り替えパネル

プレイヤーごとに戦闘設定を切り替えられます。

| 設定 | 効果 |
|------|------|
| PvP ON | 他プレイヤーと戦闘可能 |
| PvP OFF | 他プレイヤーから攻撃されず、攻撃もできない |
| PvE ON | Mobとの戦闘可能 |
| PvE OFF | Mobの攻撃を受けず、攻撃も不可（爆発も無効化） |

📌 **開き方：**  
メインパネル → PvP / PvE 切り替え  
または `/pv` コマンドを使用。

:::caution
PvEをOFFにすると、Mob戦闘だけでなく、TNTやベッド爆発も無効になります。
:::

## よくある質問

| 質問 | 回答 |
|------|------|
| パネルが開かない | コンパスを持って右クリックするか、`/cp main` を実行してください。 |
| 特定のパネルが使えない | 対応していないワールドか、権限が不足している可能性があります。 |
| 通報の返信はどこで見れる？ | ログイン時やパネル内に表示されます。 |
