# Javaパネルの使い方

24sanサーバーでは、Java版プレイヤー向けに複数のGUIパネルが提供されています。  
ここでは、各パネルの使い方を切り替えて確認できます。

---

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs queryString="panel" className="tabs-java-panels">

<TabItem value="main" label="メインパネル" default>
<div>

<h2>🧭 メインパネル</h2>
<p>すべてのパネルの起点となるパネルです。以下の機能にアクセスできます：</p>
<ul>
  <li>ギルド管理</li>
  <li>テレポート機能</li>
  <li>ポイントショップ</li>
  <li>通報システム</li>
  <li>PvP / PvE 設定切り替え</li>
</ul>
<p><strong>📌 開き方：</strong><br />
コンパスを右クリック または <code>/cp main</code> を実行</p>

</div>
</TabItem>

<TabItem value="guild" label="ギルドパネル">
<div>

<h2>🛡️ ギルドパネル</h2>
<p>ギルドの作成・参加・管理ができるパネルです。</p>
<ul>
  <li>ギルドの作成（名前・カラー・公開設定）</li>
  <li>ギルドメンバーの確認</li>
  <li>ギルドへの参加 / 脱退</li>
</ul>
<p><strong>📌 開き方：</strong><br />
メインパネル → ギルドパネル</p>

</div>
</TabItem>

<TabItem value="teleport" label="テレポートパネル">
<div>

<h2>🧭 テレポートパネル</h2>
<p>プレイヤーの移動をサポートするパネルです。</p>
<ul>
  <li>スポーン地点へ移動</li>
  <li>ギルド本拠地へ移動</li>
  <li>ワープポイントへの移動（許可された地点のみ）</li>
</ul>
<p><strong>📌 開き方：</strong><br />
メインパネル → テレポートパネル</p>

</div>
</TabItem>

<TabItem value="shop" label="ショップパネル">
<div>

<h2>🛒 ショップパネル</h2>
<p>ポイントを使用してアイテムやサービスを購入できます。</p>
<ul>
  <li>アイテム購入（ツール・装備など）</li>
  <li>特別なサービス（ワープ券・称号など）</li>
</ul>
<p><strong>📌 開き方：</strong><br />
メインパネル → ショップパネル</p>
<p>💰 所持ポイントは <code>%pointshop%</code> で確認できます。</p>

</div>
</TabItem>

<TabItem value="report" label="通報パネル">
<div>

<h2>🚨 通報パネル</h2>
<p>荒らしや不具合などを簡単に報告できるパネルです。</p>
<ul>
  <li>荒らし / バグ報告フォーム</li>
  <li>報告した内容の履歴確認</li>
  <li>通報に対する返信確認</li>
</ul>
<p><strong>📌 開き方：</strong><br />
メインパネル → 通報パネル<br />
💬 または <code>/rpt create</code> コマンドでも通報可能です。</p>

</div>
</TabItem>

<TabItem value="versu" label="PvP-PvEパネル">
<div>

<h2>⚔️ PvP / PvE 切り替えパネル</h2>
<p>プレイヤーごとに戦闘設定を切り替えられます。</p>

<table>
  <thead>
    <tr><th>設定</th><th>効果</th></tr>
  </thead>
  <tbody>
    <tr><td>PvP ON</td><td>他プレイヤーと戦闘可能</td></tr>
    <tr><td>PvP OFF</td><td>他プレイヤーから攻撃されず、攻撃もできない</td></tr>
    <tr><td>PvE ON</td><td>Mobとの戦闘可能</td></tr>
    <tr><td>PvE OFF</td><td>Mobの攻撃を受けず、攻撃も不可（爆発も無効化）</td></tr>
  </tbody>
</table>

<p><strong>📌 開き方：</strong><br />
メインパネル → PvP / PvE 切り替えパネル<br />
または <code>/pv</code> コマンドを使用</p>

<Admonition type="caution" title="注意">
PvEをOFFにすると、Mob戦闘だけでなく、TNTやベッド爆発も無効になります。
</Admonition>

</div>
</TabItem>

</Tabs>
