# 🛡️ ギルド機能まとめ

24sanサーバーでは、プレイヤー同士でチームを組める「ギルドシステム」が利用可能です。  
このページでは、ギルドに関する主要コマンドと、各機能の使い方ページへのリンクを紹介します。

---

## 📘 主要コマンド一覧

| コマンド | 説明 |
|---------|------|
| `/guild create &lt;名前&gt; &lt;カラーコード&gt; &lt;true|false&gt;` | ギルドを作成します（公開・非公開設定可能） |
| `/guild join &lt;ギルド名&gt;` | ギルドに参加します（公開ギルドのみ） |
| `/guild leave` | 現在のギルドから脱退します |
| `/guild home` | ギルド拠点へテレポートします |
| `/guild sethome` | 現在地をギルドの拠点に設定します |
| `/guild chat &lt;メッセージ&gt;` | ギルドメンバーだけのチャットを送信します |
| `/guild info` | 所属ギルドの情報を表示します |
| `/guild list` | 公開ギルドの一覧を表示します |
| `/guild invite &lt;プレイヤー名&gt;` | プレイヤーをギルドに招待します（非公開ギルド用） |
| `/guild kick &lt;プレイヤー名&gt;` | メンバーをギルドから除外します（管理者のみ） |
| `/guild disband` | ギルドを解散します（作成者のみ） |

## 💡 備考

- ギルド名やカラーコードには一部使用できない文字があります。
- ギルドの公開設定は作成時にのみ設定可能で、あとから変更できません。
- ギルドごとに PvP 設定やテリトリー制御など、今後のアップデートにより追加予定です。
