# 🛡️ ギルド機能まとめ

24sanサーバーでは、プレイヤー同士でチームを組める「ギルドシステム」が利用可能です。  
このページでは、ギルドに関する主要コマンドと、各機能の使い方ページへのリンクを紹介します。

---

## 📘 主要コマンド一覧

- `/guild` は `/g` に省略できます

| コマンド | 説明 |
|---------|------|
| `/guild create ギルド名` | ギルドを作成します |
| `/guild join ギルド名` | ギルドに参加します（公開ギルドのみ） |
| `/guild leave` | 現在のギルドから脱退します |
| `/guild warp 0-9` | ギルド拠点へテレポートします |
| `/guild enderchest` | 所属ギルドの情報を表示します |
| `/guild accept ギルド名` | 招待を受け入れる |
| `/guild deny ギルド名` | 招待を拒否する |

## 📘 ギルドオーナー向けコマンド一覧

- `/guild` は `/g` に省略できます

| コマンド | 説明 |
|---------|------|
| `/guild create ギルド名` | ギルドを作成します |
| `/guild setwarp 0-9 ワープ名` | ギルドに参加します（公開ギルドのみ） |
| `/guild disband` | 現在のギルドを削除します |
| `/guild invite プレイヤー名` | ギルド拠点へテレポートします |
| `/guild kick プレイヤー名` | ギルドから対象プレイヤーをkickします |
| `/guild ban プレイヤー名` | ギルドから対象プレイヤーをbanします |
| `/guild unban プレイヤー名` | ギルドから対象プレイヤーをban解除します |

## 💡 備考

- ギルド名やカラーコードには一部使用できない文字があります。
- ギルドの公開設定は作成時にのみ設定可能で、あとから変更できません。
- ギルドごとに PvP 設定やテリトリー制御など、今後のアップデートにより追加予定です。
