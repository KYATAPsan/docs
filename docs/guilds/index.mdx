# 🛡️ ギルド機能まとめ

24sanサーバーでは、プレイヤー同士でチームを組める「ギルドシステム」が利用可能です。  
このページでは、ギルドに関する主要コマンドと、各機能の使い方ページへのリンクを紹介します。

---

## 📘 主要コマンド一覧

:::info

`/guild` は `/g` に省略できます。  
どちらでも同じように使用可能です。

:::

| コマンド | 説明 |
|---------|------|
| `/guild create ギルド名` | ギルドを作成します |
| `/guild join ギルド名` | ギルドに参加します（公開ギルドのみ） |
| `/guild leave` | 現在のギルドから脱退します |
| `/guild warp 0-9` | ギルド拠点へテレポートします |
| `/guild enderchest` | ギルド共有エンダーチェストを開きます |
| `/guild accept ギルド名` | 招待を受け入れます |
| `/guild deny ギルド名` | 招待を拒否します |

---

## 📘 ギルドオーナー向けコマンド一覧

:::tip

以下のコマンドは、**ギルドオーナーのみ実行可能**です。  
通常メンバーは使用できません。

:::

| コマンド | 説明 |
|---------|------|
| `/guild create ギルド名` | ギルドを作成します |
| `/guild name 新ギルド名` | ギルド名を変更します |
| `/guild color カラーコード6桁` | ギルドカラーを変更します |
| `/guild setwarp 0-9 ワープ名` | ギルドワープを設定します |
| `/guild disband` | ギルドを解散します |
| `/guild setowner プレイヤー名` | ギルドオーナーを変更します |
| `/guild invite プレイヤー名` | プレイヤーを招待します |
| `/guild kick プレイヤー名` | メンバーをキックします |
| `/guild ban プレイヤー名` | メンバーをBANします |
| `/guild unban プレイヤー名` | BANを解除します |
