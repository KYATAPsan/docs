# 🎨 ギルドカラーコード一覧

ギルドの色は、カラーコードを指定して変更できます。

---

## 🛠 カラー変更コマンド

:::info

/guild color カラーコード6桁  
例: /guild color FF0000

:::

- `#` を付けずに入力してください（`FF0000`など）
- 英大文字または小文字の 0〜9、A〜F を使用してください

---

## ⚠ 注意事項

:::danger

- 白（FFFFFF）は背景と重なって見えにくくなる場合があります  
- ギルドごとに個性のある色を選ぶようにしましょう

:::
