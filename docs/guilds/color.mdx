# 🎨 ギルドカラーコード一覧

ギルドはカラーコードを指定して色を設定できます。  
このページでは、カラーの設定コマンドと、代表的な色のコード一覧を紹介します。

---

## 🛠 設定コマンド

:::info

ギルドのカラーは以下のコマンドで変更できます：

/guild color カラーコード6桁

例:  
/guild color FF0000

:::

:::tip

カラーコードには `#` を付けないでください。  
6桁の英数字（0-9, A-F）で構成された **16進数のコード** を使用します。

:::

:::danger

視認性の低い色（例: 白背景に白）は使用を避けてください。  
ギルドメンバーや他プレイヤーにとって見えづらくなる可能性があります。

:::

---

## 🎨 代表的なカラーコード一覧

| 色名     | カラーコード | 説明例（GUIで表示される色） |
|----------|--------------|------------------------------|
| 赤       | FF0000       | 強く目立つ色です             |
| 青       | 0000FF       | 安定感のある標準的な青       |
| 緑       | 00FF00       | ナチュラルで柔らかい緑       |
| 黄色     | FFFF00       | 明るく注意を引く色           |
| 紫       | 800080       | 落ち着いた高貴な色           |
| 水色     | 00FFFF       | さわやかな印象               |
| オレンジ | FFA500       | 暖かく元気な色               |
| ピンク   | FFC0CB       | 柔らかく親しみやすい色       |
| 白       | FFFFFF       | 背景と同化しやすいので注意   |
| 黒       | 000000       | 非常に目立ちます             |
| グレー   | 808080       | 中間色として使いやすい       |

---

## 💬 補足と推奨

:::info

選んだカラーコードは、ギルド名の表示やメニュー内の強調など、さまざまな場面で視覚的に使われます。

:::

:::tip

被りにくく、かつ見やすい色を選ぶことで、ギルドの印象がよくなります。

:::
